Internet

Интернет — это глобальное информационное пространство, хранящее огромное количество информации на миллионах компьютеров, которые обмениваются данными. Данные передаются по проводам, часть из которых лежит на дне океана, они называются подводными коммуникационными кабелями. В общей сложности, их длина превышает сотни тысяч миль.
